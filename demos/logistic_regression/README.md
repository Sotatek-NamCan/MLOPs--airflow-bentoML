# Logistic Regression Demo (Iris)

Dataset: Iris flower dataset.

Target (prediction): `species` (setosa, versicolor, virginica).

Features used:
- `sepal_length`
- `sepal_width`
- `petal_length`
- `petal_width`

Notes:
- All features are numeric; labels are categorical strings.
- The BentoML service expects numeric feature rows in the same order as above.
